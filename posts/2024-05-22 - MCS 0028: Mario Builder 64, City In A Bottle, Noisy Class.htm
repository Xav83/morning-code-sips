Mario Builder 64

Mario Builder 64 is a Super Mario 64 ROM hack that allows you to create custom levels in-game. You can also find the [Youtube video](https://www.youtube.com/watch?v=MUFL5ObkxCY&list=PLaUXZ0xl9h1U4e4s1tvT_0iazHfA2HB_m&index=9) to go along with the new release of the project. Can wait to try it out !!

City In A Bottle: A 256 Byte Raycasting System

This article is about a small snippet of code which is able to render a city with only a 256 byte HTML file. A really impressive performance ! And it allowed me to encounter the work of Frank Force, who have done a lot of other small programs really mesmering to watch.

Noisy: The Class You Wrote a Hundred Times

If, like me, you are a C++ developer, you have probably already written a similar class, in order to track which constructors are used, and how many times. And if we can start all using the same class to do it, we will probably improve our debugging experience as developers, since we will all have the same denomination for it. All that thanks to Vincent Zalzal!
