Cppcheck 2.14.0

CppCheck is a tool that can analyse C++ code and indicates if some issues are present. A new release is available, and 2 new checks are now present: - eraseIteratorOutOfBounds: warns when erase() is called on an iterator that is out of bounds - returnByReference: warns when a large class member is returned by value from a getter function

Docsify-this: Your website using Markdown

This open-source web app provides a quick way to publish one or more online Markdown files as standalone web pages without needing to set up your own website.
