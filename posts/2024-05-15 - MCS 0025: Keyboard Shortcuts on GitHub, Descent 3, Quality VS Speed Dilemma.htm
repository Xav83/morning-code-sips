Descent 3 Engine Open Sourced

For context, Descent 3 is a first-person shooter video game developed by Outrage Entertainment, published by Interplay Entertainment and was originally released for Microsoft Windows in North America on June 17, 1999. And the code of the game engine has been open sourced last month, giving us a look about how this game was made. Since them, some people have worked on it in order to make it available to everyone on all platform.

The Quality VS Speed Dilemma

The Dilemma of 'Quality VS Speed Dilemma' is the hypothesis stating that you can either build a quality software slowly or build a software quickly but with a poor quality of code. This article defends the idea that this is actually a false dilemma. It states that if you write code with good quality, you will iterate quickly on your code whereas if you write poor quality code, making changes will be slower and slower over time. Personnally, I tend to agree with this article.
